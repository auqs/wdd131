Overview

Purpose

The purpose of this site is to create a page that identifies if a password is strong or not. This is a great way to introduce this site plan since I am majoring in cybersecurity.

Audience

My audience would be people that are concern about their password strength. The audience would also be people that want to inform themselves of what makes a strong password, and the importance of keeping passwords up to date.

Dynamic elements

Real time passsword strength, a meter bar that tells the user whether the password is good or not and small features like adding a show/hiding pasword toggle. These dynamic elements would be used like an event listener that sees the pass and whether if it's a strong pass or not.
